## Description
Helps a user gain music recommendations according to their mood.
The app has sliders which asks the users certain questions. This data is taken by the recommendation system
which has a chatbot interface and then recommends a song to the user based on the preferences and mood that they entered

## Installation
pip install pandas numpy scikitlearn streamlit joblib matplotlib streamlit_flow nltk

## Data
https://www.kaggle.com/datasets/undefinelin/million-song-dataset-spotify-lastfm

## Basic Usage

Step 1: Open the terminal

Step 2: using cd and the path go the App folder in the projectdirectory '/Assistant_Systems/ai-assistant-project-rajeev-mangalapalli/App/'

Step 3: To run the app type in the command -> streamlit run app.py (Ensure all the installation requirements are met)

This will open the web page.
The user can then navigate to the chatbot interface throught the navigation side bar, to get song recommendations
The other pages give and overview behind the logic of the chatbot and recommendation system.
The wiki has the statistical analysis of the project and talks the user through the decisions made.
